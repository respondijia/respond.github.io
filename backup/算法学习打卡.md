还有一道题则是多数元素，找到绝对众数，这道题用到摩尔投票算法比较简单，摩尔投票算法学习链接：https://zhuanlan.zhihu.com/p/387744743。
也是利用了双指针来解决问题。
摩尔投票算法实现代码如下：
```java
int count = 0;
        int m = 0;
        for(int i = 0; i < nums.length; i++){
            if(count == 0){
                m = nums[i];
            }
            if(nums[i] == m){
                count++;
            }else{
                count--;
            }
        }
        return m;
```
